# Chuck Norris Joke Generator

A simple JavaScript application that fetches random Chuck Norris jokes from the Chuck Norris API.

This was my first project using an external API, helping me understand asynchronous JavaScript, HTTP requests, and dynamic DOM updates.

## Features

- Generate random Chuck Norris jokes
- Fetch data from a public REST API
- Loading indicator while fetching jokes
- Error handling for failed requests
- Display jokes dynamically without reloading the page

## Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Fetch API (or XMLHttpRequest if that's the version you built)
- Chuck Norris API

## How It Works

1. User clicks the "Generate Joke" button.
2. The application sends a request to the Chuck Norris API.
3. A loading message is displayed while waiting for the response.
4. Once the request completes, the joke is displayed.
5. If something goes wrong, an error message is shown.
